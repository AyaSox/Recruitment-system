? WHY THIS FIXES IT:
???????????????????????????????????????????????????????????????????

When you login, the backend sends:
  ? JWT Token with roles: ["Applicant"]
  ? User data: { email, firstName, lastName, roles }

This gets stored in browser's localStorage.

BUT... if you have OLD cached data from before, the browser uses the
old data instead of the new data!

Clearing cache = Browser gets fresh data with correct roles!
