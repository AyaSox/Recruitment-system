## ?? Available Charts

| Chart | Type | Shows | Location |
|-------|------|-------|----------|
| **Department Jobs** | Bar | Total & active jobs per dept | Dept Analytics Tab |
| **Dept Applications** | Bar | Applications per department | Dept Analytics Tab |
| **Application Status** | Pie | Status distribution | Distributions Tab |
| **Employment Type** | Pie | Job type breakdown | Distributions Tab |
| **Experience Level** | Pie | Experience breakdown | Distributions Tab |

## ?? Quick Actions

### View Charts (30 seconds)
```
1. Login as Admin/Recruiter
2. Click "Reports" in sidebar
3. Click "Department Analytics" tab
```

### Export Data (10 seconds)
```
1. Go to Reports page
2. Click "Export to Excel" button
3. Open downloaded file
```

### Create Test Data (5 minutes)
```
1. Create 8-10 jobs across different departments
2. Add 3-5 applications per job
3. Refresh Reports page
4. Charts will populate automatically
```

```
Employment Type:    Green palette
Experience Level:   Blue palette
```

## ?? Troubleshooting

| Problem | Solution |
|---------|----------|
| No charts visible | Check if logged in as Admin/Recruiter |
| "No data" message | Create jobs and applications first |
| Charts not loading | Check browser console (F12) for errors |
| Export not working | Make sure you have data to export |
