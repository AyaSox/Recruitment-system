    },

  },

  palette: {
    mode: 'dark',
    primary: {
      main: '#90caf9', // ? Change this
    },

      main: '#f48fb1', // ? Change this
    },
    // ...
  },

### **Add More Theme Options** (e.g., High Contrast):

```typescript
// In ThemeContext.tsx
type ThemeMode = 'light' | 'dark' | 'highContrast';

const highContrastTheme = createTheme({
  palette: {
    mode: 'dark',
    primary: {
      main: '#ffff00', // Yellow
    },
    background: {
      default: '#000000', // Pure black
      paper: '#000000',
    },
    text: {
      primary: '#ffffff', // Pure white
    },
  },
});
```

## ?? **TROUBLESHOOTING**

### **Theme Toggle Not Visible**:
```
1. Hard refresh browser: Ctrl+Shift+R
2. Check if Navbar.tsx is updated
3. Check browser console for errors
4. Verify ThemeToggle.tsx exists
```

### **Theme Not Persisting**:
```
1. Check browser localStorage:
   - F12 ? Application ? Local Storage
   - Should see key: "theme" with value: "light" or "dark"
2. Clear localStorage and try again
3. Check ThemeContext.tsx has useEffect for saving
```

### **Language Selector Not Working**:
```
1. Check if languages are configured in backend
2. Check LocalizationService.ts
3. Check if translations exist
4. Verify API endpoint: /api/localization/languages
```
