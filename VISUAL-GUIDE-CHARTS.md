### Desktop (1200px+)
- Charts side-by-side (2 columns)
- Full legends and labels
- Larger chart sizes

### Tablet (768px - 1199px)
- Single column layout
- Slightly smaller charts
- Abbreviated labels if needed

### Mobile (< 768px)
- Stacked layout
- Compact charts
- Rotated labels
- Touch-friendly tooltips

## ?? Usage Tips

### For Recruiters
1. Check Department Analytics to see which teams need support
2. Monitor application status distribution for bottlenecks
3. Track average applications per job to gauge job attractiveness

### For Managers
1. Compare department performance metrics
2. Identify hiring trends by employment type
3. Use charts in presentations to stakeholders

### For Admins
1. Export data for detailed analysis
2. Share visual insights with leadership
3. Track long-term recruitment trends
